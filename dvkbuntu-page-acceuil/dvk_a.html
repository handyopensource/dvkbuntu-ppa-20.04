Commandes à taper dans un Terminal

Effectuer les mises à jour disponibles

sudo apt-get update
sudo apt-get upgrade


Chercher un paquet ou une application/jeu
( nom-du-paquet est à remplacer par le nom d'application ou mot-clé de votre choix )

sudo apt search nom-du-paquet


Installer un paquet

sudo apt install nom-du-paquet


Désinstaller un paquet

sudo apt autoremove nom-du-paquet


Nettoyer la liste des paquets disponibles

sudo apt autoclean


Restaurer le dépot de dvkbuntu de la liste des dépots suite à une suppression
( adresse du dépot à mettre à jour avec l'outil Stacer )

ppa:dvkbuntu/dvkbuntu-ppa


Restaurer le dépot de dvkbuntu ou l'installer sous ubuntu ou dérivées

sudo add-apt-repository ppa:dvkbuntu/dvkbuntu-ppa
sudo apt install dvkbuntu


Désinstaller le dépot de dvkbuntu

sudo apt autoremove --purge dvkbuntu
sudo add-apt-repository --remove ppa:dvkbuntu/dvkbuntu-ppa
