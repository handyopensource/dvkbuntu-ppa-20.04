Commands to type in a Terminal

Perform available updates

sudo apt-get update
sudo apt-get upgrade


Search for a package or an application/game
( nom-du-paquet est à remplacer par le nom d'application ou mot-clé de votre choix )

sudo apt search nom-du-paquet


Installing a package

sudo apt install nom-du-paquet


Uninstalling a package

sudo apt autoremove nom-du-paquet


Clean the list of available packages

sudo apt autoclean


Restore the dvkbuntu repository from the repository list after deletion
(address of the repository to be updated with the Stacer tool)

ppa:dvkbuntu/dvkbuntu-ppa


Restore the dvkbuntu repository or install it under ubuntu or derivatives

sudo add-apt-repository ppa:dvkbuntu/dvkbuntu-ppa
sudo apt install dvkbuntu


Uninstall the dvkbuntu package repository

sudo apt autoremove --purge dvkbuntu
sudo add-apt-repository --remove ppa:dvkbuntu/dvkbuntu-ppa
