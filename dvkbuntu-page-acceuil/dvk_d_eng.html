Kubuntu is a free operating system of the GNU/Linux type. It is a project to use the KDE graphical environment instead of Unity and then Gnome within Ubuntu. The Kubuntu project is not a distribution derived from Ubuntu, because both share exactly the same base, the same software, the same APT repositories, the same code name and the same development cycle.
Like Ubuntu, Kubuntu is simple to install, getting to the point, and easily detecting devices. In short, Kubuntu is to Ubuntu what KDE is to Unity.
Since April 2004, Kubuntu was an official project sponsored by Canonical Ltd. Since April 2012, Canonical no longer supports this project. The German company Blue Systems GmbH has since taken on the role of main sponsor.

According to the Kubuntu Project website FAQ, the term kubuntu is a word meaning "for humanity" in Bemba; it is pronounced /kùbúntú/ (kou-BOUN-tou). But others consider it more logical to see the addition of the K prefix (from KDE) to the Bantu word ubuntu, meaning "humanity to others".
